Setup Python

Welcome to the Python Tutorial. This guide will help you get started with Python programming language.

Getting Started

  1. Create a Folder for the Tutorial Open your terminal and create a new folder for the tutorial: bash mkdir ~/Tutorial/Macbook-Python cd ~/Tutorial/Macbook-Python

    1. Install Python If you don't have Python installed, download and install it from the official website:

      • Download Python bash brew install python

    2. Install Visual Studio Code Download and install Visual Studio Code (VS Code) for a powerful and lightweight code editor:

      • Download VS Code bash brew install --cask visual-studio-code

      After installing, you can open VS Code from the terminal: bash code .

  2. Set Up a Virtual Environment Create a virtual environment to manage your project dependencies: bash python3 -m venv venv source venv/bin/activate

  3. Install Required Packages Install any necessary packages using pip: bash pip install <package-name>

  4. Start Coding Create a new Python file and start coding: bash touch main.py open main.py

Happy coding!

Troubleshooting

Python Installation Issues

  1. Python Command Not Found If you receive a "command not found" error when trying to run Python:

    • Ensure Python is installed correctly by running: bash python3 --version
    • If not installed, reinstall Python
