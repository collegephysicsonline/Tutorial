  1. To calculate the average velocity, use the formula: [ \text{average velocity} = \frac{\text{initial velocity} + \text{final velocity}}{2} ]

    initial_velocity = 10  # m/s
final_velocity = 20  # m/s
average_velocity = (initial_velocity + final_velocity) / 2

  2. The gravitational force between two masses is given by Newton's law of gravitation: [ F = G \frac{m_1 m_2}{r^2} ] where ( G ) is the gravitational constant.

    G = 6.67430e-11  # gravitational constant
mass1 = 5.0  # kg
mass2 = 10.0  # kg
distance = 2.0  # m
gravitational_force = G * (mass1 * mass2) / (distance ** 2)

  3. The kinetic energy of an object is given by: [ KE = \frac{1}{2} m v^2 ]

    mass = 2.0  # kg
velocity = 3.0  # m/s
kinetic_energy = 0.5 * mass * (velocity ** 2)

  4. The potential energy of an object at a height is given by: [ PE = mgh ] where ( g ) is the acceleration due to gravity.

    mass = 2.0  # kg
height = 5.0  # m
g = 9.8  # m/s^2
potential_energy = mass * g * height

    force = 10.0  # N

    work_done = 100.0  # J
time_taken = 5.0  # s
power_output = work_done / time_taken

    initial_velocity = 0  # m/s

  28. The energy of a photon is given by: [ E = \frac{hc}{\lambda} ] where ( h ) is Planck's constant.

Solutions for Additional Problems

  2. To determine the factorial of a given number:

    number = 5
factorial = 1
for i in range(1, number + 1):
    factorial *= i

  3. To find the sum of all even numbers between 1 and 20:

    sum_even_numbers = sum(i for i in range(1, 21) if i % 2 == 0)

  4. To calculate the average of a list of numbers:

    numbers = [2, 4, 6, 8, 10]
average = sum(numbers) / len(numbers)

  5. To determine the maximum value in a list of numbers:

    numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5]

  6. To find the number of occurrences of the number 3 in a list:

    numbers = [3, 1, 4, 3, 5, 3, 2, 3, 5]

  7. To calculate the sum of the squares of the first 5 natural numbers:

    sum_squares = sum(i**2 for i in range(1, 6))

  8. To determine the product of all odd numbers between 1 and 10:

    product_odd_numbers = 1
for i in range(1, 11):
    if i % 2 != 0:
        product_odd_numbers *= i

  9. To find the sum of all prime numbers less than 20:

    def is_prime(n):
    if n <= 1:
        return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:
            return False
    return True

sum_primes = sum(i for i in range(2, 20) if is_prime(i))

  10. To calculate the Fibonacci sequence up to the 10th term:

    fibonacci_sequence = [0, 1]
for i in range(2, 10):
    next_term = fibonacci_sequence[-1] + fibonacci_sequence[-2]
    fibonacci_sequence.append(next_term)
